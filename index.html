Como usar

1. Entre como ADM usando admin/admin123.
2. Crie usuários na aba Usuários.
3. Vá em Arquivos e selecione uma pasta com imagens.
4. Use a busca ou filtro de pasta para encontrar imagens.
5. Baixe uma imagem, uma pasta filtrada ou todos os arquivos.


Importante: esta versão salva tudo no navegador usando IndexedDB/localStorage. Para vários computadores acessarem os mesmos arquivos, precisa conectar com Supabase Storage.
