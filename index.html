Como usar

1. Rode o SQL no Supabase e crie o bucket arquivos no Storage.
2. Faça login com admin / admin123.
3. O ADM cria usuários. O usuário comum só importa pastas e baixa arquivos.
4. Para baixar vários arquivos, o navegador pode pedir permissão para múltiplos downloads.
